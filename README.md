# employee-management-app

A simple multi-page web application for managing employee contact information, built with HTML, CSS, and JavaScript.

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with system overview and stats |
| New Contact | `new-contact.html` | Form to add a new employee |
| View Contacts | `view-contacts.html` | Table of all employees with search and filter |


## Project Structure

```
employee-management-app/
├── index.html
├── new-contact.html
├── view-contacts.html
├── styles.css
├── main.js
└── images/
```
